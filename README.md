## 📌 Project Overview

Recommendation systems are widely used in travel, e-commerce, and entertainment platforms to provide personalized suggestions based on user preferences. This project develops a **Tourist Recommendation System** that recommends tourist destinations using **Collaborative Filtering** and **Cosine Similarity**.

The system analyzes user ratings and destination information to generate personalized travel recommendations, helping users discover places that match their interests.

---

## 🎯 Project Objectives

* Develop a personalized tourist recommendation system.
* Analyze tourism data using Exploratory Data Analysis (EDA).
* Build a User-Item Interaction Matrix.
* Generate destination recommendations using Collaborative Filtering.
* Measure similarity between users and destinations using Cosine Similarity.
